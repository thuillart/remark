# Node.js

Learn how to send your first feedback using the Node.js SDK.

## Prerequisites

To get the most out of this guide, you'll need to:

- [Create an account](/sign-up)
- [Create an API key](/api-keys)

## 1. Install

Get the Node.js SDK using your preferred package manager:

```bash
// @filename="Terminal"
npm install @nucleon-sh/sdk
```

## 2. Send a test feedback

```ts
// @filename="server.ts"
import { Nucleon } from "@nucleon-sh/sdk";

const nucleon = new Nucleon("nu_123456789");

(async function () {
  const { data, error } = await nucleon.feedbacks.send({
    from: "delivered@nucleon.sh",
    text: "Hello, world!",
  });

  if (error) {
    return console.error({ error });
  }

  console.log({ data });
})();
```
