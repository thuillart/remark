## Description of the Service

remark.sh is a SaaS platform that aims to help product builders collect, organize and act on user feedback. Core features include:

- A lightweight integration (SDK or API) for submitting feedback
- Automated de-duplication and categorization powered by a large language model (LLM)
- An online dashboard for voting, filtering and tracking feedback items

Please note: remark.sh automates feedback management but does not replace professional consulting or support services.

<section id="how-it-works" className="scroll-mt-20">

## How It Works

When a user submits a feedback, our SDK sends it securely to us. We then process it with a large language model (LLM) to remove duplicates, categorize it and group similar feedback.

For A/B testing, we may switch between models (e.g., Google Gemini, OpenAI ChatGPT), but we never use your feedback data to train those external services, nor do we sell it.

You’ll then see everything in your dashboard (feedback, votes, categories, contacts, etc.).
